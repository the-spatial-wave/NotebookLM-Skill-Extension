# TSW COURSE_CREATOR PROMPTS

## 3.7 — Curriculum Map
Scrivi una CURRICULUM MAP completa in italiano seguendo il template: 1) Intestazione Corso (Nome, Piattaforma: Skool, Prezzo, Durata, North Star), 2) Mappa Moduli (N, Titolo, Tipo, Output studente, Tempo), 3) Struttura per ogni modulo (Intro, Contenuto, Output, Risorse), 4) Sequenza Email collegata. Basati sulle fonti per strutturare il percorso dal dolore al risultato.

## 3.7 — Lesson Scripts
Genera gli script dettagliati per le lezioni del Modulo 1 e 2. Tono: formativo, pratico, incoraggiante. Ogni script deve avere: Hook (perché guardare), Core Content (cosa imparare), Action Step (cosa fare ora).

## 3.7 — Worksheets
Crea 3 worksheet/esercitazioni pratiche in PDF (formato markdown per ora) che portino lo studente a produrre il primo output tangibile del corso. Ogni worksheet deve avere: Obiettivo, Istruzioni, Esercizio guidato.

## 3.7 — Skool Setup
Genera una guida tecnica per configurare la community Skool: 1) Categorie Classroom consigliate, 2) Sistema di Gamification (Gems/Levels) basato sull'impegno, 3) Messaggio di benvenuto automatico personalizzato sul topic.
